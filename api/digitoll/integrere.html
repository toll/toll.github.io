Integrere

Plassholder: felles integrasjonsguide. Stegene er like for norske og utenlandske aktører; der det er forskjeller markeres de inline.

Forskjell norsk / utenlandsk:
Plassholder for markør: brukes gjennom siden til å fremheve hvor norske og utenlandske aktører avviker (typisk scope-navn og sertifikat).

Forutsetninger

Plassholder: du har et gyldig sertifikat (se «Bestille sertifikat») og har fullført onboarding (se «Onboarding») – integrasjon opprettet hos Digdir med tildelt scope.

1. Klientintegrasjon

Plassholder: applikasjonen henter token og veksler til Toll-signert access token.

1.1 Hent access token fra Maskinporten

Plassholder: lenke til Digdirs dokumentasjon. Markør for forskjell: norske aktører vs. European eSeals.

1.2 Veksle til et Toll-signert access token

Plassholder: HTTP-request (POST), header, body, eksempel-respons.

1.3 Test tilgang til API-et

Plassholder: test-auth-kall, forventet 200 OK.

Scopes

Plassholder: oversikt over scope. Markør for forskjell: norske scope (toll:movement/...) vs. utenlandske (toll:eu/movement/...).

Drift og retry

Plassholder: anbefaling om caching av Toll-token og retry-mekanisme.
